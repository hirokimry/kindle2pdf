## データ保護

### 機密情報の取り扱い

- シークレット・APIキー・パスワードをリポジトリにコミットしない
- 環境変数またはシークレットマネージャーで管理する
- Google Cloud Vision の API キーは環境変数 `GOOGLE_VISION_API_KEY` からのみ取得し、コード・設定にコミットしない。鍵不在で `google` を選んだ場合は黙って進めず、OCR 段開始時に明確なエラーで停止する（fail-closed・Issue #56）

### データ送信・第三者連携

- **既定はローカル完結（外部送信なし）**。撮影・OCR・PDF 化はすべて端末内で行う。
- 例外は OCR エンジンに `google`（Google Cloud Vision）をオプトイン選択した場合のみで、そのとき **書籍ページ画像が Google のサーバーへ HTTPS 送信される**（従量課金・Issue #56）。既定の `apple`（Apple Vision）では送信は発生しない。
