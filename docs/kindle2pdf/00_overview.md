# 📚 Kindle本PDF化システム ドキュメント一覧

購入済みKindle本を、Mac上でフル自動でスクショ→OCR→**検索可能PDF**化するシステムの設計資料一式です。

## 📄 ファイル構成

| ファイル | 内容 |
|---------|------|
| `00_README.md` | この目次 |
| `01_仕様・設計書.md` | 要件・技術選定・アーキテクチャ・停止条件・**実機PoC検証結果**（v0.1 確定版） |
| `02_実装計画書.md` | ファイル構成・モジュールAPI・主要コマンド・座標変換・PoCマイルストーン（Claude Code着手用） |
| `PoCサンプル_検索可能.pdf` | PoCで生成した検索可能PDF（4枚のスクショ→左右分割→OCR→8ページ） |

## ✅ 現在のステータス

計画・仕様策定・設計フェーズは**完了**。実機PoCで主要な設計判断を実データ検証済み。

- OCR = **Apple Vision**（`ocrmac`）で確定
- 撮影 = **`screencapture` CLI**（サムネ写り込み回避）
- ページ化 = **見開き左右分割**、停止判定 = **pHash（距離≤2＝同一）**
- Vision座標→PDF透明テキスト層は**Y反転不要**と確定
